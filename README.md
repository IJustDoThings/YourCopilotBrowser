# YCB Browser

A fast, clean browser built on WebView2 with Copilot integration.

🌐 **[ycb.tomcreations.org](https://ycb.tomcreations.org)**

---

## Features

- 🤖 Built-in GitHub Copilot CLI integration
- 🔒 Password manager with encryption
- 🕵️ Incognito mode
- 📌 Bookmarks & history
- 🔔 Chrome-style permission prompts (camera, mic, location)
- ⚙️ Per-site settings & cookie controls
- 📥 Download manager
- 🌙 Dark UI

## Requirements

- Windows 10/11
- [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)
- [.NET 8 Runtime](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
